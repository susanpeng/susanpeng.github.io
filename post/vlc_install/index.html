Install VLC Media Player

2022-02-10

Contents

VLC Media Player is an application that lets you easily play music, watch videos, and browse photos on your computer. These instructions will show you how to download it from the www.cnet.com website and install it on your Mac.

Find VLC Media Player

  1. Open the browser you prefer to use.

  2. Type www.cnet.com in the address bar to open the website.

  3. On the menu bar, hover your mouse over Download. You will see five items in the drop-down menu.


    aaa

  4. Click Mac Software. You will see a page with VLC Media Player on the right side of the page.

Download VCL Media Player

  1. Click VLC Media Player. You will see the download page for VLC Media Player.

  2. Click . Downloading starts.

You can monitor the downloading progress with the indicator in the bottom left corner of the screen.

Start In progress Done

Install VLC Media Player

  1. Click the down arrow next to the file. A pop-up menu displays.

  2. Click Show in Finder. A finder window opens with the downloaded file named vlc-2.1.0.dmg inside.

  3. Double click the file named vlc-.2.1.0.dmg. The VLC Media Player starts installing. When the installation is finished, a start window displays.

  4. Double click VLC icon. A dialogue box displays to ask if you want to open the VLC Media Player.

  5. Click Open button. The VLC Media Player will start.

Congratulations! You have successfully downloaded and installed VLC Media Player on your Mac.

Now, you are able to use VLC Media Player to play music, watch videos, and browse photos.
